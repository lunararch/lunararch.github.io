# Pastel Color Palette

## Primary Colors
- Pastel Magenta: #F8A5C2 (RGB: 248, 165, 194)
- Light Purple: #C5A3FF (RGB: 197, 163, 255)
- Light Pink: #FFD1DC (RGB: 255, 209, 220)
- Light Blue: #A5DEF1 (RGB: 165, 222, 241)

## Secondary Colors
- Soft Mint: #C3E8BD (RGB: 195, 232, 189)
- Pale Yellow: #FFF0B5 (RGB: 255, 240, 181)
- Light Lavender: #E0D3FF (RGB: 224, 211, 255)
- Soft Peach: #FFDAB9 (RGB: 255, 218, 185)

## Background Colors
- Off-White: #FAFAFA (RGB: 250, 250, 250)
- Light Gray: #F0F0F0 (RGB: 240, 240, 240)
- Dark Text: #333333 (RGB: 51, 51, 51)
- Medium Text: #666666 (RGB: 102, 102, 102)

## Accent Colors
- Darker Magenta: #E06C9F (RGB: 224, 108, 159)
- Darker Purple: #9F7AEA (RGB: 159, 122, 234)
- Darker Blue: #76C1D4 (RGB: 118, 193, 212)
- Soft Shadow: rgba(0, 0, 0, 0.05)

## Gradient Combinations
- Magenta to Purple: linear-gradient(to right, #F8A5C2, #C5A3FF)
- Blue to Purple: linear-gradient(to right, #A5DEF1, #C5A3FF)
- Pink to Peach: linear-gradient(to right, #FFD1DC, #FFDAB9)
- Lavender to Mint: linear-gradient(to right, #E0D3FF, #C3E8BD)

## Usage Guidelines
- Use primary colors for main UI elements and highlights
- Use secondary colors for supporting elements and variety
- Use background colors for text backgrounds and containers
- Use accent colors for hover states, buttons, and important elements
- Use gradients for section backgrounds or decorative elements
- Maintain sufficient contrast between text and background colors for accessibility
